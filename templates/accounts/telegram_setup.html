Настройка Telegram авторизации

Внимание! Для работы авторизации через Telegram необходимо создать бота.

1 Создание бота

Перейдите к @BotFather в Telegram и создайте нового бота:

Открыть @BotFather
  • Отправьте команду /newbot
  • Введите имя бота (например: "Construction Manager Bot")
  • Введите username бота (например: "construction_manager_bot")
  • Скопируйте полученный токен

2 Настройка через команду

Выполните команду с вашими данными:

python manage.py setup_telegram_bot --token YOUR_BOT_TOKEN --username YOUR_BOT_USERNAME

3 Настройка через .env файл

Или добавьте настройки в .env файл:

TELEGRAM_BOT_TOKEN=ваш_токен_бота TELEGRAM_BOT_USERNAME=имя_вашего_бота

4 Перезапуск сервера

После настройки перезапустите сервер:

python manage.py runserver

5 Тестирование

Перейдите на страницу авторизации и попробуйте войти через Telegram:

Попробовать авторизацию
Важно! Убедитесь, что username бота указан без символа @ (например: "my_bot", а не "@my_bot").
