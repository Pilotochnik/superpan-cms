Как работают ключи доступа:
  1. Выберите проект - к которому нужно предоставить доступ
  2. Выберите пользователя - кому предоставляется доступ
  3. Добавьте описание - для чего предоставляется доступ
  4. Система автоматически создаст уникальный ключ доступа
Важно знать:
  • Каждый пользователь может иметь только один активный ключ к проекту
  • Ключи можно деактивировать в любой момент
  • Пользователь получит доступ к канбан-доске и может добавлять расходы
